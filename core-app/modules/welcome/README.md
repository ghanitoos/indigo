Welcome module

Provides a simple welcome page for new employees. Features:
- LDAP lookup by username or email
- Manual gender selection
- Password generation for PC, Email and Cloud according to local rules
- Independent module scaffold (routes, templates, static)

Permissions:
- Requires permission `welcome.access` to view and use endpoints

Installation:
- Add `welcome` to `ENABLED_MODULES` env variable or register module in app
